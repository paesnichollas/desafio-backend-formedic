# 01 - Visão geral

Este projeto entrega uma API backend mínima para agendamentos.

Escopo funcional:

- Criar agendamento
- Listar por dia e profissional
- Bloquear conflito de horário

Não inclui:

- Autenticação/autorização
- Cancelamento ou remarcação
- Regras avançadas de agenda
- Componentes distribuídos
